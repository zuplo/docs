Zuplo is a fully-managed, programmable API gateway for developers. Build,
secure, and scale APIs with authentication, rate limiting, analytics, and more.
Deploy globally in seconds with Git-based workflows.

## Get started

New to Zuplo? Start with the quickstart to have your first API gateway running
in minutes.

- [Quickstart (Portal)](./articles/step-1-setup-basic-gateway.md) - Set up a
  gateway using the Zuplo Portal
- [Quickstart (Local)](./articles/step-1-setup-basic-gateway-local.md) - Set up
  a gateway using local development
- [What is Zuplo?](./api-management/introduction.md) - Learn how Zuplo fits into
  your stack

## Core features

- [Policies](./policies/overview.md) - Add authentication, rate limiting,
  validation, and more to your API with pre-built or custom policies
- [Handlers](./handlers/url-forward.md) - Route requests to backends, Lambda
  functions, or custom code
- [API Keys](./articles/api-key-management.md) - Built-in API key management
  with self-serve developer access
- [Monetization](./articles/monetization/index.md) - Charge for API usage with
  built-in billing and subscription management

## Products

- [AI Gateway](./ai-gateway/introduction.md) - Manage multiple AI providers
  through a single API with observability, rate limiting, and cost controls
- [MCP Server](./mcp-server/introduction.md) - Connect your APIs to AI
  assistants using the Model Context Protocol
- [Developer Portal](./dev-portal/introduction.md) - Create interactive API
  documentation with built-in testing and self-serve API key management
