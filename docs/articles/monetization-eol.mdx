Zuplo's v1 monetization product has reached end of life. This decision reflects
a commitment to delivering the best possible experience for API monetization
while focusing on core strengths - building the best API management product.

## Important Timeline

:::danger

**The monetization service was shut down on November 15th, 2025.**

No new projects can be created with the legacy monetization features.

:::

## Why This Decision Was Made

After careful consideration and extensive customer feedback, the v1 monetization
product was sunset for the following reasons:

### Limited Monetization Strategies

Our v1 monetization product lacked the flexibility many customers needed,
including:

- Pay-as-you-go pricing models
- Credit-based systems
- Usage-based billing with variable pricing tiers
- Custom pricing models for enterprise customers

### Customer Feedback

Customer feedback indicated a need for a complete monetization solution that
includes:

- Comprehensive UI for managing the entire API monetization lifecycle
- Self-service customer portals for subscription management
- Built-in support for upgrades, downgrades, and plan changes
- Streamlined refund and credit management
- Advanced analytics and revenue reporting
- Multi-currency support
- Tax calculation and compliance features

### Focus on Core Strengths

Zuplo's strength lies in building the most powerful, developer-friendly API
management platform. By focusing on API management rather than billing and
payments, Zuplo delivers even more value in the core product.

## Zuplo Monetization

Zuplo has since launched a new, fully integrated
[Monetization](./monetization/index.mdx) product that addresses all of the
limitations of v1. The new product provides built-in metering, quota
enforcement, Stripe billing integration, and a self-serve Developer Portal — all
native to the gateway.

## Migration path

1. **Shutdown complete** — The v1 monetization service was shut down on November
   15, 2025.
2. **Migrate to Zuplo Monetization** — Follow the
   [Quickstart](./monetization/quickstart.md) to set up the new product.
3. **Contact support** — If you need migration assistance, reach out to
   support@zuplo.com.
